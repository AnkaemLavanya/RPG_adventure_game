# RPG_adventure_game
A console-based RPG game developed using Python where the player battles randomly generated enemies. The player can attack, use potions to heal, or run away to survive. The game uses loops, dictionaries, conditional statements, and the random module to simulate combat mechanics and create an interactive adventure experience.

# Text-Based RPG Adventure Game Using Python

## 📌 Overview

This is a simple **console-based RPG (Role Playing Game)** built using Python.
The player fights a randomly selected enemy and must decide whether to attack, use healing potions, or run away. The game continues until the player defeats the enemy, escapes, or loses all health.

## 🎮 Features

* Random enemy selection
* Turn-based combat system
* Attack damage calculation
* Healing using potions
* Run away option
* Game over and victory conditions
* Interactive console gameplay

## 🛠️ Technologies Used

* Python 3
* Random module
* Dictionaries and lists
* Loops and conditional statements

## ▶️ How to Run

1. Install **Python 3** on your system.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```
python rpg_game.py
```

## 🎯 Gameplay Instructions

1. Enter your **hero name** when the game starts.
2. A random enemy will appear.
3. Choose an action:

   * **1 → Attack** the enemy
   * **2 → Run Away**
   * **3 → Use Potion** to restore health
4. Survive the battle and defeat the enemy to win.

## 💡 Learning Concepts
